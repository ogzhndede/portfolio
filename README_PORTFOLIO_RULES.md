# Portfolio Rules Pack

This folder contains planning and rules files for the portfolio website.

Use these files with Codex or another AI coding agent to keep implementation consistent.

## Recommended Placement

Copy these into the root of the portfolio repository:

```text
portfolio/
  AGENTS.md
  rules/
    project-rules.md
    design-rules.md
    content-data-rules.md
    playable-system-rules.md
    deployment-rules.md
    codex-task-plan.md
```

## How to Start with Codex

First prompt:

```text
Read AGENTS.md and all files under /rules. Then inspect the current project. Do not change code yet. Summarize the current stack, routing approach, styling approach, and GitHub Pages readiness.
```

Then continue with tasks from:

```text
rules/codex-task-plan.md
```

## Important

This rules pack is for the portfolio website only.
